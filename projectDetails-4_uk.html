Slim Mom (командний)

Командний проект Slim Mom, це додаток, для схуднення. Надає доступ до калькулятора ваги/калорій, а також дає змогу розробити свою дієту для схуднення. В проекту я займався фомою для введення своїх даних в калькулятор розрахунку калорій. Також приймав участь в розробці бокового меню, яке відображає кількість спожитих/добову норму калорій. Проект також реалізований на трьох мовах: англійська, українська, французька.

Форма для авторизації

Форма для авторизації містить поля для пошти та паролю, всі поля введення валідуються.

Головна сторінка

Головна сторінка розділена на дві секції зліва календар з покупками на кожен день, справа калькулятор калорій, які вираховуються з списку страв, які можна ввести у пошуку.

Головна сторінка французькою

Головна сторінка французько - всі написи на сайті переведені на французьку та змінюються перемикачем у хедері

Головна сторінка англійською

Головна сторінка англійською - всі написи на сайті переведені на англійську та змінюються перемикачем у хедері

Форма для введення своїх фізичних даних

Користувач вводить свої фізичні дані для розрахунку денної норми калорій. Форма включає в себе: зріст, вік, вагу, бажану вагу, групу крові. Всі поля валідуються.

  • react

    Бібліотека JavaScript для створення інтерфейсів користувача, яка дозволяє будувати компоненти та ефективно управляти станом для динамічних веб-додатків.

  • formik

    Бібліотека для управління станом та валідацією форм у React, яка спрощує роботу з формами та їх обробку.

  • yup

    Бібліотека для валідації даних у JavaScript, яка дозволяє створювати схеми для перевірки форм та об'єктів на відповідність визначеним правилам.

  • axios

    Бібліотека для управління станом у JavaScript/React-додатках, яка забезпечує автоматичне відстеження змін у даних і оновлення інтерфейсу користувача.

  • redux

    Бібліотека для управління станом у JavaScript-додатках, яка використовує єдиний глобальний стан і дозволяє керувати ним через функції-диспатчери і редюсери.

  • react-router-dom

    Бібліотека для реалізації маршрутизації в React-додатках, яка дозволяє створювати маршрути і управляти навігацією між різними компонентами та сторінками.

  • prop-types

    Бібліотека для перевірки типів властивостей (props) у компонентах React, що допомагає забезпечити коректність даних, що передаються в компоненти.

  • react-i18next

    Бібліотека для міжнародизації в React-додатках, яка дозволяє легко додавати підтримку кількох мов і керувати перекладами.
