SPYRO GYM

Лендінг SPYRO GYM, містить в собі основну інформацію про заклад, дає можливість залишити персональну інформацію користувача для подальшого зв'язку, або записатись на пробне тренування. Розробка проекту підвищила мої уміння у верстці, стилізаціх та css-анімаціях. Також я отримав цінний досвід роботи з сервісом онлайн баз даних QuintaDB. Також цінним уроком для мене була імплементація google-maps у проект.

Головне зображення

Містить слоган та коротку рекламу тренажерного залу. При натисканні кнопки 'Start free trail' відкривається модальне вікно з формою для заповнення персональних даних. В хедері розташована кнопка 'Contacts' з модальним вікном, яке містить всю необхідну інформацію для того щоб зв'язатись/дістатись з тренажерним залом.

Список інструкторів SPYRO GYM

Список інструкторів містить в собі фотографію кожного інструктора, при натисканні на яку, висвітлюється коротке біо тренера. На менших розширеннях девайсів повний список приховано, він розгортається після натискання на кнопку.

Список занять у SPYRO GYM

Список містить коротку інформацію про кожен урок: тип, кількість учасників, назву, час, номер заняття, рівень складності. Кожен ітем списку має дві опції - дізнатись більше та запис на пробне заняття. При натисканні на кнопку 'дізнатись більше', відкриється модальне вікно з короткою інформацією про заняття та графіком занять. При кліку на кнопку 'почати пробне заняття' відкриється модальне вікно у якому буде фома для заповнення персональних даних з можливістю зміни складності заняття. Список можна розгорнути щоб побачити всі доступні уроки.

Коротка інформація про заняття

При натисканні на кнопку 'дізнатись більше' в списку занять, відкриється модальне вікно з короткою інформацією про заняття та графіком занять.

Основні напрямки діяльності тренажерного залу

SPYRO GYM має 4 основних напрямки діяльності: тренажерний зал, бойові мистецтва, зумба, йога. При наведені на віконце напрямку, воно розширяється і даже короткий опис напрямку, більш об'ємний опис доступний нижче, при кліку на анімоване віконце напряму.

Секція відгуків

Слайдер з відгуками відвідувачів містить в собі: короткий відгук, оцінку, ім'я та фото користувача.

  • react

    Бібліотека JavaScript для створення інтерфейсів користувача, яка дозволяє будувати компоненти та ефективно управляти станом для динамічних веб-додатків.

  • formik

    Бібліотека для управління станом та валідацією форм у React, яка спрощує роботу з формами та їх обробку.

  • yup

    Бібліотека для валідації даних у JavaScript, яка дозволяє створювати схеми для перевірки форм та об'єктів на відповідність визначеним правилам.

  • axios

    Бібліотека для управління станом у JavaScript/React-додатках, яка забезпечує автоматичне відстеження змін у даних і оновлення інтерфейсу користувача.

  • swiper

    Бібліотека для створення сучасних і чутливих слайдерів та каруселей на веб-сторінках, з можливістю налаштування та підтримкою різних ефектів переходу.

  • react-youtube

    Бібліотека для вбудовування та управління YouTube відео в React-додатках, яка надає простий API для взаємодії з YouTube плеєром.

  • react-google-maps

    Бібліотека для інтеграції Google Maps у React-додатки, що спрощує роботу з картами, маркерами та іншими елементами Google Maps API.

  • dotEnv

    Бібліотека для завантаження змінних середовища з файлу .env у Node.js додатки, що дозволяє зберігати конфіденційні дані окремо від коду.

  • prop-types

    Бібліотека для перевірки типів властивостей (props) у компонентах React, що допомагає забезпечити коректність даних, що передаються в компоненти.
